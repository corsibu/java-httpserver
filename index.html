The Problem

Java doesn't really have a good, easy to implement and start using, HTTP server. Those that do exist don't have easy access to all of the underlying data provided with an HTTP request. As such, we put together our own system for Storyteller, extracted the HTTP bits, and polished it up a whole bunch.

So... What is it?

This is a more polished version of the original underlying HTTP server used by Storyteller. By more polished, we mean that there's been a good deal of added features to make it easy for people to start using quickly. The main goals of this server are to permit quick creation and to provide users with a largely customizable system.

How can I use it?

That might be better answered in one of two places:

  • java-httpserver's javadoc
  • Our simple tutorial

You'll also need Java 7, mostly because we use multi-catch blocks, which aren't supported before Java 7.
