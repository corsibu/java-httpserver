Getting Started

java-httpserver is still under development, and needs to be finalized a little more, but at present, the easiest way to get started is to download the httpserver directory from the main git repository, and include it in the root of your project.

Following that, you need to create at least one new class, an HTTPHandlerFactory, which is used to determine what kind of HTTPHandler should be used to handle an incoming request.

A basic server

If you want to see a working example, look at the demo directory on the git repository.

A very basic server driver looks something like this: 

// file: Driver.java
import httpserver.*;

public class Driver {
    public static void main(String[] args) {
        HTTPServer server = new HTTPServer();
        server.setHandlerFactory(new MyHandlerFactory());

        server.run();
    }
}
// file: MyHandlerFactory.java
import httpserver.*;

public class MyHandlerFactory extends HTTPHandlerFactory {
    public MyHandlerFactory() {}

    public HTTPHandler determineHandler(String pathSegment, HTTPRequest request) throws HTTPException {
        if (checkIfEquals(pathSegment, "hello", request))
            return new HelloHandler(request.getConnection(), request);

        // MessageHandler comes free with the httpserver package
        return new MessageHandler(request, 418, "Nothing to do");
    }
}
